== Description ==

SacchaOne is a lightweight, easily customizable WordPress theme. It’s perfect for blogs, small business, startups, agencies, firms as well as personal portfolio sites and most types of projects.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "SacchaOne"
3. Install and activate
